1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?

ans : getElementById("id") নির্দিষ্ট ID দিয়ে ১টি element return করে। getElementsByClassName নির্দিষ্ট class দিয়ে multiple elements return করে । querySelector CSS selector ব্যবহার করে প্রথম matching element return করে। querySelectorAllCSS selector ব্যবহার করে সব matching elements return করে ।

2. How do you create and insert a new element into the DOM?

Ans : document.createElement() দিয়ে element তৈরি করা হয়। content যোগ করা হয় (innerText, innerHTML)। DOM-এ insert করা হয় (appendChild, prepend, append, etc.)

3. What is Event Bubbling and how it works?
Ans : Event bubbling হলো যখন কোনো element-এ event trigger হয়, তখন eventটি child থেকে parent  grandparent document পর্যন্ত উপরে propagate করে।

4. What is Event Delegation and why it is useful?

Ans : Event delegation হলো parent element-এ event listener বসিয়ে তার child elements-এর event handle করা।
performance better 
dynamically added elements এর জন্যও কাজ করে

5. Difference between preventDefault() and stopPropagation()?

Ans : preventDefault()  browser-এর default action বন্ধ করে (যেমন form submit reload হওয়া বন্ধ)।
stopPropagation() event bubbling বন্ধ করে (parent-এ event যেতে দেয় না)।
